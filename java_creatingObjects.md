### BUILD JAVA APP ###
```java
        public class HelloWorld {
                public static void main(String[] args) { 
                        System.out.println("Hello World"); 
                }
        }               
```

        The file needs to have same name as the class
        
        :compile:        javac HelloWorld.java
        :run:            java HelloWorld
