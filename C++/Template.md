## USAGE:
	### TEMPLATE FUNCTION
	* Def: Template function is function who can take
				 typename (int,double,float,char ...) as parameters
				 and  return Typename T.
						 
	* Example:
```cpp
	template<typename T,size_t const size>
	T compute(T& a, T&b)
	{
		return a + b;
	}
```
